# Xampp on Ubuntu 18.04

PHP files can be placed on:

    /opt/lampp/htdocs/


Commands:

	sudo /opt/lampp/lampp stop
	sudo /opt/lampp/lampp start

To uninstall:

	cd  /opt/lampp
	sudo  ./uninstall
