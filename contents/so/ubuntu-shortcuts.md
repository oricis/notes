# Keyboard shortcuts (used in Ubuntu)

> Open terminal:

	Ctrl + Alt + T

> Swap between the open applications and documents:

	Alt + Tab or Super + Tab

> Toggle between desktop or open applications and documents:

	Super + D

> Close tabs in the current application, f.e. web browser tabs:

	Ctrl + W

> Close the application window:

	Alt + F4
